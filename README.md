# Student Feedback Form - DevOps CA-2

## Description
This project is a Student Feedback Registration Form developed using HTML, CSS, and JavaScript. The form is tested using Selenium (Java) and automated using Jenkins.

## Technologies Used
- HTML
- CSS
- JavaScript
- Selenium (Java)
- Maven
- Jenkins

## Test Cases Implemented
- Valid form submission
- Invalid email validation
- Empty field validation
- Button functionality testing

## Automation
Jenkins is used to automate the execution of Selenium test cases using Maven.

## Output
- Form UI
- Automated testing using Selenium
- Jenkins Build Success
